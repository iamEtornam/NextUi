# Typography (Text)

A comprehensive text widget that provides both traditional heading styles (h1-h6) and Material Design 3 typography scale.

## Traditional Headings

Quickly apply standard heading styles with weight variants:

```dart
// Bold headings
NextText.h1Bold('Main Heading')
NextText.h2Bold('Sub-heading')
NextText.h3Bold('Section Title')
NextText.h4Bold('Subsection')
NextText.h5Bold('Minor Title')
NextText.h6Bold('Small Title')

// Medium weight headings
NextText.h1Medium('Medium Weight H1')
NextText.h2Medium('Medium Weight H2')

// Regular weight headings
NextText.h1Regular('Regular Weight H1')
NextText.h2Regular('Regular Weight H2')
```

## Material Design 3 Scale

Follow the official Material 3 typography guidelines:

```dart
// Display - Largest text on screen
NextText.displayLarge('Display Large')
NextText.displayMedium('Display Medium')
NextText.displaySmall('Display Small')

// Headline - High-emphasis text
NextText.headlineLarge('Headline Large')
NextText.headlineMedium('Headline Medium')
NextText.headlineSmall('Headline Small')

// Title - Medium-emphasis text
NextText.titleLarge('Title Large')
NextText.titleMedium('Title Medium')
NextText.titleSmall('Title Small')

// Body - Regular body text
NextText.bodyLarge('Body Large')
NextText.bodyMedium('Body Medium')
NextText.bodySmall('Body Small')

// Label - Call-to-action text
NextText.labelLarge('Label Large')
NextText.labelMedium('Label Medium')
NextText.labelSmall('Label Small')
```

## Body Text Variants

Styles for paragraphs, descriptions, and captions:

```dart
// Paragraph styles
NextText.paragraphBold('Bold paragraph text')
NextText.paragraphMedium('Medium paragraph text')
NextText.paragraphRegular('Regular paragraph text')

// Description styles
NextText.descriptionBold('Bold description text')
NextText.descriptionMedium('Medium description text')
NextText.descriptionRegular('Regular description text')

// Caption styles
NextText.captionBold('Bold caption text')
NextText.captionMedium('Medium caption text')
NextText.captionRegular('Regular caption text')

// Subheading styles
NextText.subHeadingBold('Bold subheading')
NextText.subHeadingMedium('Medium subheading')
NextText.subHeadingRegular('Regular subheading')

// Extra small styles
NextText.xsmBold('Bold extra small text')
NextText.xsmMedium('Medium extra small text')
NextText.xsmRegular('Regular extra small text')
```

## Customization

Easily override colors and other text properties:

```dart
NextText.h1Bold(
  'Colored Text',
  textColor: Colors.blue,
  textAlign: TextAlign.center,
)

NextText.bodyLarge(
  'Limited text content...',
  maxLines: 1,
  overflow: TextOverflow.ellipsis,
)
```

## Basic Usage

You can also use the base NextText widget for custom styling:

```dart
NextText(
  'Custom styled text',
  style: TextStyle(fontSize: 16, fontWeight: FontWeight.bold),
  textColor: Colors.red,
  textAlign: TextAlign.center,
)
```

## API Reference

### Text Static Methods

Next UI Text provides static methods for all typography levels. Each method accepts the following optional parameters:

| Parameter | Type | Description |
| --- | --- | --- |
| `textColor` | `Color?` | Color override |
| `textAlign` | `TextAlign?` | Text alignment |
| `maxLines` | `int?` | Maximum number of lines |
| `overflow` | `TextOverflow?`| Overflow behavior |
