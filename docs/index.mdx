# Introduction

Next UI is a comprehensive, customizable Flutter component library inspired by HeroUI (formerly NextUI) and designed to follow modern design patterns.

It provides a set of high-quality widgets that are easy to use, highly customizable, and responsive.

## Features

- **Modern Design**: Inspired by HeroUI's sleek and modern aesthetics.
- **Customizable**: Extensive properties to tweak every aspect of the components.
- **Type Safe**: Built with Dart's strong type system.
- **Responsive**: Components adapt well to different screen sizes.
- **Clean Architecture**: Follows best practices for internal organization.

## Quick Start

To use Next UI in your Flutter project, follow the [Getting Started](/getting-started) guide.

## Components

Next UI comes with a rich set of components:

- **Layout**: Card, Table, Modal
- **Data Display**: Avatar, Badge, Chip, Text, Tooltip
- **Inputs**: Button, Checkbox, Input, Radio Group, Switch
- **Feedback**: Circular Progress, Toast
- **Navigation**: Dropdown
